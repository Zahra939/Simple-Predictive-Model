# Simple Predictive Model Using Python

## Objective
Build a machine learning model to predict passenger survival using the Titanic dataset.

## Tools Used
- Python
- Pandas
- Scikit-Learn

## Algorithm
Logistic Regression

## Steps Performed
1. Loaded dataset
2. Cleaned missing values
3. Converted categorical data into numerical form
4. Split data into training and testing sets
5. Trained Logistic Regression model
6. Evaluated model performance

## Result
Successfully built a predictive model with approximately 79% accuracy on the Titanic dataset.
